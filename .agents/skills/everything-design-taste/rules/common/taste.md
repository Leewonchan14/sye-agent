# Taste Rules

1. **No defaults.** Every font, color, spacing value, and layout choice must be intentional. If you can't explain why, change it.

2. **No AI aesthetic convergence.** Never produce: Inter/Roboto defaults, purple gradients on white, three-column card grids with drop shadows, centered hero sections with stock imagery.

3. **Hierarchy is mandatory.** Every screen must have a clear visual hierarchy. Squint test: can you see 3 distinct levels of importance with eyes unfocused?

4. **Less is more, but empty is nothing.** Minimalism is a choice. Laziness looks like minimalism but isn't. The difference is intentionality.

5. **Consistency beats cleverness.** A consistent 7/10 design system beats brilliant individual screens that don't work together.

6. **Constraints create quality.** Don't fight constraints. Use them to make better decisions.

7. **Test your assumptions.** "I think users will..." is a hypothesis, not a fact. Treat it accordingly.
