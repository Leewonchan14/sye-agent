# Hierarchy Rules

1. Every page has exactly one primary action. If you have two, you have zero.
2. Visual weight communicates importance. The most important element should be the most prominent.
3. Use at least two differentiation methods between hierarchy levels (size + weight, color + spacing).
4. Group related items with proximity. Separate unrelated items with space.
5. Reading order follows visual weight: largest/boldest first, smallest/lightest last.
6. Navigation should never compete with content for attention.
7. Labels, captions, and metadata are supporting actors. Style them accordingly (smaller, lighter, less contrast).
8. When everything is bold, nothing is bold.
