# Brand Consistency Rules

1. Reference brand colors by semantic token name, never by hex value directly.
2. Use brand fonts as specified. Substitutions require explicit approval.
3. Logo usage: maintain minimum clear space, never distort, never recolor without guidelines approval.
4. Voice and tone: match the documented brand voice attributes in all copy output.
5. Photography: follow the documented style guide for subject, lighting, and color treatment.
6. Icons: use the brand icon set exclusively. Don't mix icon libraries.
7. Motion: follow documented timing and easing curves. Don't introduce new motion patterns without system review.
8. Error handling: use brand voice in all error messages, not generic language.
9. Brand elements should feel natural, not forced. If a brand application feels awkward, the guidelines may need updating, not just enforcement.
10. Document deviations. If a project needs to diverge from brand guidelines, document why and get approval.
