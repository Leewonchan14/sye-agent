# Retro Computing Design System

## Color Tokens
```css
:root {
  --color-bg-primary: #1E1E2E;
  --color-bg-secondary: #282840;
  --color-bg-tertiary: #313150;
  --color-text-primary: #CDD6F4;
  --color-text-secondary: #A6ADC8;
  --color-accent: #89B4FA;
  --color-accent-hover: #74C7EC;
  --color-border: #45475A;
  --color-success: #A6E3A1;
  --color-warning: #F9E2AF;
  --color-error: #F38BA8;
}
```

## Typography
- Display: "Space Mono", monospace (400, 700)
- Body: "IBM Plex Sans", sans-serif (400, 500)
- Mono: "Space Mono", monospace
- System text in mono. Readable text in sans.

## Spacing
4 / 8 / 12 / 16 / 24 / 32 / 48

## Radius
Small: 4px | Medium: 8px (keep it tight)

## Borders
1px solid borders are a primary design element. Use them freely. Double borders for emphasis.

## Motion
- Micro: 100ms linear
- Standard: 200ms ease-out
- Consider step-based animations (not smooth) for character

## Personality
The charm of early computing without being a theme park. Monospace typography as a statement. Window chrome and borders as structural elements. Dense information display. The feeling of a tool built by someone who cares about precision. Not nostalgic, not ironic. Respectful.
